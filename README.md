- **Objetivo:** Criar uma Visão Cliente, Visão Comportamento, Visão Campanhas e Visão Ponto de Vendas.  
- **Principais insights:**  
  - Top 5 produtos mais lucrativos

  - Tendência de crescimento em regiões específicas
